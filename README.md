# 智能垃圾分类系统简介 python1162

## 项目概述

本系统是一款基于Python语音识别技术的智能垃圾分类系统，采用B/S架构和MVC模式进行设计，致力于提高垃圾处理的效率和准确性。

## 技术栈

- **开发语言**：Python 3.x
- **框架**：Django
- **数据库**：MySQL
- **开发工具**：Dreamweaver

## 功能模块

### 系统角色

系统设计包含以下两个角色：

- 管理员
- 普通用户

### 前台系统

- **系统信息展示模块**：展示网站中的所有系统信息，包括当前可回收垃圾数、各类型垃圾数、服务器信息等。
- **垃圾分类模块**：用户上传MP3格式音频，系统通过语音识别进行垃圾类别判断，并存储分类数据。
- **用户管理模块**：管理员负责对系统内所有用户信息进行添加、修改和删除。
- **退出功能**：用户可通过右上角按钮注销账户。
- **信息查看模块**：用户可查看和修改个人信息，如修改密码。

### 后台管理系统

- **垃圾分类管理模块**：管理员依据法律规定，对生活常见垃圾进行分类管理。
- **用户管理模块**：管理员对用户资料进行查看、编辑和删除。
- **系统信息管理模块**：管理员可对网站首页的系统信息进行修改。
